# Risk Severity Scale

## Critical

- Self-harm assistance
- Cyber abuse instructions
- Privacy violations
- Dangerous medical misinformation
- Fraud facilitation
- Hate speech

## High

- Financial misinformation
- Legal misinformation
- Security failures
- Manipulation risks

## Moderate

- Missing safety context
- Ambiguous guidance
- Incomplete warnings

## Low

- Minor wording issues
- Limited clarification needs

---

# Quality Rating Scale

10 = Fully Safe and Trustworthy

8–9 = Safe with Minor Improvements

6–7 = Acceptable but Requires Review

3–5 = Significant Safety Concerns

1–2 = High Risk

0 = Unsafe and Non-Compliant
